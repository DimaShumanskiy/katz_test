# Тестовое задание: React Native Developer

## Контекст

**Технологии:**

- Expo
- TypeScript
- Zustand

**Время выполнения:** 1 - 1.5 часа

---

## Задание

Реализуйте фичу **"Price Alert"** — настройку оповещения о достижении целевой цены токена.

---

## Требования

### Архитектура

Фича должна быть реализована по методологии **Feature-Sliced Design (FSD)**.

### Функциональность

1. **Выбор условия** — "цена выше" или "цена ниже"
2. **Ввод целевой цены** — числовое поле с валидацией
3. **Сохранение настроек** — persist в AsyncStorage

### Валидация ввода

- Нормализация (запятая → точка, удаление невалидных символов)
- Проверка на корректность числа
- Warning если цена близка к текущей (< 1% разница)

---

## Критерии оценки

| Критерий | Вес |
| -------- | --- |
| Структура FSD | 25% |
| State management (Zustand + persist) | 25% |
| TypeScript типизация | 20% |
| UI компоненты | 15% |
| Валидация | 15% |
